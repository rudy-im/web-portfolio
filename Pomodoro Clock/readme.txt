In command line, move to the project folder.

> cd <path>

For external drive and path with blanks, adjust command a bit.

> cd /d "<path>"



Then run any kind of server.

> py -m http.server <port-number>   // python 3 server

> npx serve .   // Node.js server - Port 3000



If you are editing the files and refresh is not working properly due to the cache,

Ctrl + Shift + R
